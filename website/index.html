Features

Everything your folders
deserve

Set it up once. Shelve runs quietly in the background and keeps things tidy — across as many folders as you want.

Watch any folder
Monitor Downloads, Desktop, Documents, or any custom path — add as many as you want from the setup wizard or Settings tab.
Plain-English rules
Type "trash installers bigger than 200MB" and Shelve creates the rule instantly — no dropdowns required.
Powerful conditions
Mix file type, keywords, date, time of day, size, name pattern, and internet-download flag into a single precise rule.
TF-IDF search
Instantly find any file across all watched folders with a relevance-ranked search engine built entirely in Swift.
Auto-rename
Add date prefixes, lowercase filenames, swap spaces for underscores, or add custom prefixes — applied before moving.
History & undo
Every move is logged. One click in the menu bar undoes the last action — no digging through folders required.
Smart Rule Creator

Rules in plain English

Type what you want. Shelve figures out the conditions, extensions, and destination automatically.

✨ "trash installers bigger than 200MB"
→ Trash rule · .dmg .pkg · size > 200 MB
✨ "move PDFs older than 30 days to Archive"
→ Archive folder · .pdf · older than 30 days
✨ "files downloaded between midnight and 3am"
→ TimeFiles folder · time of day 0:00–3:00
✨ "partial or failed downloads"
→ FailedDownloads · .crdownload .part .tmp
✨ "images from the internet larger than 5MB"
→ WebDownloads · .jpg .png · web flag · >5 MB
✨ "scripts ending with .sh to Code"
→ Code folder · name ends with .sh

All parsing is done locally — no API keys, no internet connection needed.
