Features

Everything your Downloads
folder deserves

Set it up once. Shelve runs quietly in the background and keeps things tidy.

Auto-classify
Watches your Downloads folder and moves files into the right subfolder automatically — PDFs, images, code, videos, and more.
Custom rules
Create your own rules with any combination of file types, keywords, date conditions, file size, and name patterns.
TF-IDF search
Instantly find any organized file with a relevance-ranked search engine built entirely in Swift. No indexing delay.
Auto-rename
Add date prefixes, lowercase filenames, swap spaces for underscores, or add custom prefixes — applied before moving.
Move to Trash
Rules can trash files instead of sorting them. Perfect for auto-deleting old installers, temp files, and failed downloads.
History & undo
Every move is logged. One click in the menu bar undoes the last action — no digging through folders required.
